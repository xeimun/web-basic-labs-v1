색상과 배경 속성

CSS에서는 다양한 방법으로 색상을 지정하고 배경을 설정할 수 있습니다. 이 페이지에서는 색상 값의 표현 방식, 배경색, 배경 이미지, 그라데이션 등을 학습합니다.

색상 값 표현 방식

1. 색상 이름

color: red;
color: green;
color: blue;
color: yellow;
color: purple;

CSS는 140개 이상의 표준 색상 이름을 지원합니다.

2. RGB/RGBA 값

color: rgb(255, 0, 0);     /* 빨강 */
color: rgb(0, 255, 0);     /* 녹색 */
color: rgb(0, 0, 255);     /* 파랑 */
color: rgba(255, 0, 0, 0.5); /* 반투명 빨강 */
color: rgba(0, 0, 255, 0.5); /* 반투명 파랑 */

RGB는 Red, Green, Blue의 혼합으로 색상을 표현합니다. 각 값은 0에서 255 사이입니다. RGBA는 알파(투명도) 채널을 추가한 것으로, 0(완전 투명)에서 1(완전 불투명) 사이의 값을 가집니다.

3. HEX 코드

color: #ff0000;  /* 빨강 */
color: #00ff00;  /* 녹색 */
color: #0000ff;  /* 파랑 */
color: #ff6600;  /* 주황 */
color: #6600cc;  /* 보라 */

16진수 표기법으로, #RRGGBB 형식으로 표현합니다. 각 색상 쌍은 0에서 255를 16진수(00-FF)로 나타냅니다.

4. HSL/HSLA 값

color: hsl(0, 100%, 50%);     /* 빨강 */
color: hsl(120, 100%, 50%);   /* 녹색 */
color: hsl(240, 100%, 50%);   /* 파랑 */
color: hsla(0, 100%, 50%, 0.5);   /* 반투명 빨강 */
color: hsla(240, 100%, 50%, 0.5); /* 반투명 파랑 */

HSL은 Hue(색상, 0-360), Saturation(채도, 0%-100%), Lightness(명도, 0%-100%)로 색상을 표현합니다. HSLA는 알파 채널을 추가한 것입니다.

배경 속성

1. 배경색 (background-color)

.bg-color-example {
    background-color: #3498db;
}

2. 배경 이미지 (background-image)

.bg-image-example {
    background-image: url('https://hips.hearstapps.com/hmg-prod/images/alpe-di-siusi-sunrise-with-sassolungo-or-langkofel-royalty-free-image-1623254127.jpg?crop=1xw:1xh;center,top&resize=500:*');
    height: 150px;
}

3. 배경 반복 (background-repeat)

repeat (기본값)

repeat-x

repeat-y

no-repeat

.bg-repeat-example {
    background-image: url('pattern.png');
    background-repeat: repeat;
}
.bg-repeat-x-example {
    background-image: url('pattern.png');
    background-repeat: repeat-x;
}
.bg-repeat-y-example {
    background-image: url('pattern.png');
    background-repeat: repeat-y;
}
.bg-no-repeat-example {
    background-image: url('pattern.png');
    background-repeat: no-repeat;
}

4. 배경 위치 (background-position)

center

top right

50% 25%

.bg-position-1 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: center;
}
.bg-position-2 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: top right;
}
.bg-position-3 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: 50% 25%;
}

5. 배경 크기 (background-size)

auto

cover

contain

.bg-size-1 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-size: auto;
}
.bg-size-2 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-size: cover;
}
.bg-size-3 {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-size: contain;
}

6. 배경 고정 (background-attachment)

scroll (기본값)

내용을 스크롤해보세요

배경은 스크롤됩니다

fixed

내용을 스크롤해보세요

배경은 고정됩니다

.bg-attachment-scroll {
    background-image: url('image.jpg');
    background-attachment: scroll;
}
.bg-attachment-fixed {
    background-image: url('image.jpg');
    background-attachment: fixed;
}

7. 배경 단축 속성 (background)

.bg-shorthand {
    background: #3498db url('pattern.png') no-repeat center/cover;
    /* 색상 이미지 반복 위치/크기 */
}

단축 속성으로 여러 배경 속성을 한 번에 설정할 수 있습니다.
