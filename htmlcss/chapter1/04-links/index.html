링크의 기본 구조

HTML에서 링크는 <a> 태그를 사용하여 만듭니다. 가장 중요한 속성은 href로, 이동할 URL을 지정합니다.

기본 링크 예제

Example 웹사이트

<a href="https://www.example.com">Example 웹사이트</a>

링크 유형

1. 외부 링크

Google (새 탭에서 열기)

<a href="https://www.google.com" target="_blank">Google (새 탭에서 열기)</a>

속성 target="_blank"는 링크를 새 탭이나 창에서 열도록 지정합니다.

2. 내부 링크 (같은 웹사이트 내 다른 페이지)

목록 페이지로 이동

<a href="../03-lists/index.html">목록 페이지로 이동</a>

3. 페이지 내 링크 (북마크)

링크 속성 섹션으로 바로가기

<a href="#link-attributes">링크 속성 섹션으로 바로가기</a>

4. 이메일 링크

이메일 보내기

<a href="mailto:example@example.com">이메일 보내기</a>

5. 전화번호 링크

전화 걸기: 010-1234-5678

<a href="tel:+8210-1234-5678">전화 걸기: 010-1234-5678</a>

링크 속성

속성 설명 예제
href 링크의 목적지 URL href="https://www.example.com"
target 링크가 열릴 위치 target="_blank" (새 탭/창)
rel 현재 문서와 링크된 문서 사이의 관계 rel="nofollow" (검색 엔진이 링크를 따라가지 않도록)
title 링크에 대한 추가 정보 (마우스 오버 시 툴팁) title="이 링크에 대한 설명"
download 링크를 클릭하면 대상을 다운로드하도록 지정 download="filename.pdf"

상대 경로와 절대 경로

절대 경로

전체 URL을 지정합니다.

<a href="https://www.example.com/page.html">절대 경로 예제</a>

상대 경로

현재 페이지를 기준으로 경로를 지정합니다.

  • 같은 폴더 내 파일: <a href="contact.html">연락처</a>
  • 하위 폴더 내 파일: <a href="products/item.html">제품</a>
  • 상위 폴더 내 파일: <a href="../index.html">홈으로</a>
