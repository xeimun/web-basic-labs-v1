HTML 문서의 기본 구성 요소

HTML 문서는 다음과 같은 기본 구조로 구성됩니다:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>문서 제목</title>
</head>
<body>
    <!-- 문서의 내용 -->
</body>
</html>

주요 요소 설명

<!DOCTYPE html>
문서 유형을 선언합니다. HTML5를 사용함을 브라우저에 알립니다.
<html>
HTML 문서의 루트 요소입니다. 모든 다른 HTML 요소는 이 요소 안에 위치해야 합니다.
<head>
문서에 대한 메타데이터를 포함하는 요소입니다. 제목, 스타일, 스크립트, 메타 정보 등을 정의합니다.
<meta>
문서에 대한 메타데이터를 정의합니다. 문자 인코딩, 뷰포트 설정, 설명, 키워드 등을 지정할 수 있습니다.
<title>
브라우저의 제목 표시줄이나 페이지 탭에 표시되는 문서의 제목을 정의합니다.
<body>
웹 페이지에서 실제로 보여지는 내용이 들어가는 영역입니다.

시맨틱 태그

HTML5는 웹 페이지의 콘텐츠를 더 명확하게 정의하는 시맨틱 태그를 제공합니다:

  • <header>: 문서나 섹션의 헤더 영역
  • <nav>: 탐색 링크 모음
  • <main>: 문서의 주요 콘텐츠 영역
  • <section>: 문서의 독립적인 섹션
  • <article>: 독립적인 콘텐츠 영역
  • <aside>: 주요 콘텐츠와 간접적으로 관련된 콘텐츠
  • <footer>: 문서나 섹션의 바닥글 영역
