Flexbox란?

Flexbox는 요소들을 행이나 열 방향으로 쉽게 배치할 수 있게 해주는 CSS 레이아웃 모델입니다. 복잡한 레이아웃을 보다 효율적으로 구현할 수 있습니다.

Flex 항목 속성

1. flex-grow

남는 공간을 항목들이 얼마나 차지할지 비율을 지정합니다.

1 (0)
2 (1)
3 (2)
.flex-grow-demo {
    display: flex;
}
.grow-0 {
    flex-grow: 0; /* 기본값, 늘어나지 않음 */
}
.grow-1 {
    flex-grow: 1; /* 남는 공간의 1/3 차지 */
}
.grow-2 {
    flex-grow: 2; /* 남는 공간의 2/3 차지 */
}

2. flex-shrink

공간이 부족할 때 항목들이 얼마나 줄어들지 비율을 지정합니다.

No Shrink (0)
Shrink (1)
More Shrink (2)
.flex-shrink-demo {
    display: flex;
    width: 100%;
}
.shrink-0 {
    flex-shrink: 0; /* 줄어들지 않음 */
    width: 200px;
}
.shrink-1 {
    flex-shrink: 1; /* 기본값, 표준 비율로 줄어듦 */
    width: 200px;
}
.shrink-2 {
    flex-shrink: 2; /* 다른 항목보다 2배 더 줄어듦 */
    width: 200px;
}

3. flex-basis

항목의 기본 크기를 지정합니다.

Auto
0
100px
200px
.flex-basis-demo {
    display: flex;
}
.basis-auto {
    flex-basis: auto; /* 기본값 */
}
.basis-0 {
    flex-basis: 0; /* 컨텐츠 크기 무시 */
}
.basis-100 {
    flex-basis: 100px;
}
.basis-200 {
    flex-basis: 200px;
}

4. flex 단축 속성

flex-grow, flex-shrink, flex-basis를 한 번에 지정합니다.

1
auto
none
custom
.flex-shorthand-demo {
    display: flex;
}
.flex-1 {
    flex: 1; /* flex: 1 1 0%; */
}
.flex-auto {
    flex: auto; /* flex: 1 1 auto; */
}
.flex-none {
    flex: none; /* flex: 0 0 auto; */
}
.flex-custom {
    flex: 2 0 100px; /* flex-grow: 2, flex-shrink: 0, flex-basis: 100px */
}

5. order

항목의 순서를 지정합니다.

2 (order: 1)
3 (order: 2)

    display: flex;
}
.order-1 {
    order: 1;
}
.order-2 {
    order: 2;
}
.order-3 {
    order: 3;
}
